Adds a simple textarea to posting and page editing, for internal notes.

== Description ==
Simply put, this plugin adds a simple textarea to posting and page editing where you can type internal notes and miscellaneous notes related to this page or post.

= Future Updates =
- Add an optional field to the post and page overview showing the notes there.

== Installation ==
= Automatic installation =
1. Press **Install** to install this plugin to your WordPress.
2. Enable the plugin in WP admin.

= Manual installation =
1. Download `internal-notes.zip` from any available source.
2. Connect to your FTP server and upload and extract the zip-archive to your `/wp-content/plugins/` folder.
3. Enable the plugin in WP admin.
